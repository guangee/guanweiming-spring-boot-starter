# 使用方式
在pom.xml中加入以下代码
```xml
<dependency>
    <groupId>com.github.guangee</groupId>
    <artifactId>guanweiming-spring-boot-starter</artifactId>
    <version>版本号</version>
</dependency>
```

```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>


```
