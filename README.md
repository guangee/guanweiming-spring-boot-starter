# 使用方式
在pom.xml中加入以下代码
```xml
<dependency>
    <groupId>com.guanweiming.common</groupId>
    <artifactId>guanweiming-spring-boot-starter</artifactId>
    <version>1.0.8</version>
</dependency>
```
